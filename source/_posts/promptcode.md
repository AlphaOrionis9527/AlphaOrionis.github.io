## 一、使用DeepSeek排除故障：
直接将报错信息复制粘贴给DeepSeek，输入“提示错误”+粘贴错误信息。
打开联网搜索，点击发送。
收到结果后，根据AI提示结果逐步排查即可。
code时一定注意缩进位置关系，和标点符号。
### 你会发现当你解决了你的第一个问题的时候，我们常常提到的那个“套路”一目了然了。
这种事嘛，就得“先开枪，后画靶”。

## 二、提示AI帮助你编程：
不要纠结你的编程语法基础，试着提出好的问题。
保持对知识发展的跟进，留意职业领域相关的技术演进。
多看开源项目，如果可以，参与其中，作出贡献。
思考项目解决了什么问题，还可以怎么进化？
下载好项目需要的组件，拿着开源项目的README.md，丢给DeepSeek，打开深度思考。
提示词可以这样写：
示例，注意让AI给你个进度条，让你的工作有个盼头。
``` bash
参考README.md文档，按照下面的要求为我编写程序
## 需求：
# 核心目标：我需要一个自动化处理视频的工具，用于将视频中的人物说话内容转换成文本。
# 主要功能：
- 可以读取单个或多个视频文件
- 识别提取视频中的人物说话内容
- 输出每个视频中提取内容的文本文件
- 添加进度条显示处理进度

## 输入/输出：
- 输入：用户可以直接在代码中输入一个文件夹路径，文件夹中包含单个或多个mp4视频文件
- 输出：输出单个或多个以视频文件名命名的txt文档，文档内是对应视频识别到的文字内容，保存到与输入视频相同的文件夹中。

## 环境要求：
- macOS系统兼容
- python 3.11.5

## 注意：
- 生成的代码全部使用中文注释
- 详细告诉如何安装环境
- 我已经将faster-whisper所需的模型下载到了本地电脑中，我希望可以在代码中制定模型的路径
- 我需要处理的视频是中文语言
```
第一次AI给出的代码不一定能完美运行，但你可以不断的盘它。
参考“一、使用DeepSeek排除故障”，这就是不断优化的过程。
最终优化到自己想要的结果。

## 四、Word排版提示词示例：
当你向DeepSeek提问生成文字时，经常会想怎么让AI能够给你一个直观粗暴的一键下载按钮，而不是一行一行复制粘贴。
下面这个示例是帮助你理解和解决整理文档的思路：
我们让DeepSeek根据要求生成文字，可直接在提示词中的需求进行添加修改内容；是否消除人工智能痕迹则根据文字的实际应用场景来判断；段落格式是我们想要在Word文档中呈现的版式，可根据需求进行调整；最终让AI给我们生成一个html文件供我们下载。
这里需要注意的是DeepSeek并不能直接为我们生成微软Office的Word文档，所以我们需要将上面步骤生成的html文件下载下来，然后用Word方式打开，最后再另存为Word文档。这样，我们需要的内容、版式和整体Word文件就完成了。
``` bash
## 需求：
# 消除人工智能痕迹。

# 格式和排版要求如下：
-一级标题 字体为宋体（三号），行距24磅值，段后空0.5行
-二级标题 字体为黑体（三号），行距24磅值，段后空0.5行
-正文 三号仿宋GB-2312字体，对齐方式为两端对齐，首行缩进两个字符

# 用html的格式输出，需要有能直接下载word文档的功能，并且能直接下载word文件。
```
根据示例内容可以不断调整优化。
对于办公、通知、论文等格式固定的内容，我们可以套用这种方法进行排版，避免大量重复低效的工作。
